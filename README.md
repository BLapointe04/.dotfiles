# My Dotfiles
These are my dotfile configuration files for different software in Bash.

## .nanorc
This is my custom .nanorc configuration for Nano.

## .bashrc
This is my custom .bashrc configuration for Bash.
- `bin/linux.sh`: Script to set up custom Linux environment.
- `bin/cleanup.sh`: Reverses the setup made by `linux.sh`.
- `etc/bashrc_custom`: Contains custom aliases and functions.
- `Makefile`: Automates running scripts for setup and cleanup.
